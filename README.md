# Savitur Website

Savitur Website is a responsive landing page built with HTML, CSS, and JavaScript.

The project focuses on clean layout, modern design, and adaptive interface for different screen sizes.

## Features
- Responsive layout (mobile / tablet / desktop)  
- Clean and semantic HTML structure  
- Modern CSS (Flexbox, Grid)  
- Interactive UI elements  
- Smooth scrolling and basic animations  

## Tech Stack
- HTML5  
- CSS3 (Flexbox, Grid)  
- JavaScript (ES6)  

## Live Demo

# Savitur Website

Savitur — это адаптивный лендинг, созданный с использованием HTML, CSS и JavaScript.

Проект демонстрирует чистую вёрстку, современный дизайн и адаптацию под разные устройства.

## Возможности
- Адаптивная вёрстка (мобильные / планшеты / десктоп)  
- Чистая и семантическая HTML-разметка  
- Современный CSS (Flexbox, Grid)  
- Интерактивные элементы интерфейса  
- Плавный скролл и базовые анимации  

## Технологии
- HTML5  
- CSS3 (Flexbox, Grid)  
- JavaScript (ES6)
